Place your CS104 .py files here. Example:

hello.py
calculate.py

Files will be listed on /python-projects and can be executed with the Run button.

DB project grouping:
- createDB.py: initialize the shared people.db users table
- insert_recs.py: seed or update users from users.py into the same shared database
- show_records.py: display all users from the shared database
- users_db_project.py: grouped project that runs the whole workflow in one execution

Security: Only .py files inside this folder can be executed. Execution is limited by a timeout and output length.
