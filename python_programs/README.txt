Place your CS104 .py files here. Example:

hello.py
calculate.py

Files will be listed on /python-projects and can be executed with the Run button.

DB project grouping (run in this order):
1. createDB.py - ensure the shared people.db and users table exist
2. insert_recs.py - seed default users into the shared database
3. show_records.py - display current users from the shared database
4. users_db_project.py - interactive menu for creating, seeding, updating, and viewing users

If you want a single interactive workflow, run users_db_project.py and choose options 1-4.

Security: Only .py files inside this folder can be executed. Execution is limited by a timeout and output length.
